spl new target variable name, reference name, bitcounts... create a bit splitter for a variable spl new bytes Bytes 8 8 8 8 8 8 #splits 'bytes' into 6 actual bytes
obtainf output variable, splitter reference, constant bitcount index get the range of bits of a splitter's target variable at a constant index spl obtainf a Bytes 1 #gets the 2nd range of bits in 'Bytes'
obtainv output variable, splitter reference, bitcount index get the range of bits of a splitter's target variable at an index that may be determined in runtime
this generates a lot more code than obtainf so don't use it when you don't need to define the bit range index in runtime		 spl obtainv a Bytes x #gets the x-th range of bits in 'Bytes'
clearf splitter reference, constant bitcount index clear the range of bits of a splitter's target variable at a constant index spl clearf Bytes 3 #clears the 4th range of bits in 'Bytes'
clearv splitter reference, bitcount index clear the range of bits of a splitter's target variable at an index that may be determined in runtime
this generates a lot more code than clearf so don't use it when you don't need to define the bit range index in runtime		 spl clearv Bytes x #clears the x-th range of bits in 'Bytes'
writef input, splitter reference, constant bitcount index write to the range of bits of a splitter's target variable at a constant index spl writef 0b10100101 Bytes 3 #writes a number to the 4th range of bits in 'Bytes'
writev input, splitter reference, bitcount index write to the range of bits of a splitter's target variable at an index that may be determined in runtime
this generates a lot more code than writef so don't use it when you don't need to define the bit range index in runtime		 spl writev 0xFF Bytes x #writes a number to the x-th range of bits in 'Bytes'
